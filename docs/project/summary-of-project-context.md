# Project Context Summary

This repository is the curated project memory for the Ananas AI Platform.

It captures the final distilled decisions from the design process:
- marketing-first internal AI platform
- portal on existing Ananas stack
- AWS runtime and Microsoft identity
- Teams channels and Teams bot in Phase 1
- multi-model routing for budget control
- 5 core Phase 1 agents
- stronger Phase 2 for competitor intelligence, lead-gen/outreach, self-learning memory, advanced evaluation, autonomous workflows, and video systems
