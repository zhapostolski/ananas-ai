# Naming Standard

Use lowercase kebab-case for files and folders unless the format requires otherwise.

## Active stable files
- `CLAUDE.md`
- `docs/architecture/architecture-v1.md`
- `docs/budgets/budget-v1.md`
- `docs/roadmap/phase-1-roadmap.md`
- `config/metrics.json`
- `config/agents.json`
- `config/model-routing.json`
- `config/schedules.json`
- `config/integrations-matrix.json`

## Diagram files
- `diagrams/source/architecture-v1.mmd`
- `diagrams/source/architecture-v1.dot`
- `diagrams/export/architecture-v1.svg`
- `diagrams/export/architecture-v1.png`

## Decisions
- `docs/decisions/adr-001-*.md`
- `docs/decisions/adr-002-*.md`
- etc.
