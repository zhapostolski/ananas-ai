# Repo Structure

## Goal
Provide a Claude Code-ready repository that is also clear for human contributors.

## Top-level structure
- `CLAUDE.md` project memory
- `.claude/` Claude Code project assets
- `docs/` architecture, implementation, operations, leadership, budget
- `config/` project-driving configuration
- `schemas/` JSON output contracts
- `sql/` database bootstrap and migrations
- `src/ananas_ai/` runtime starter code
- `scripts/` developer and QA helpers
- `resources/` frameworks and templates
- `examples/` sample inputs and outputs
- `diagrams/` editable and exportable architecture diagrams
