# ADR-002 Teams Bot Included in Phase 1

## Status
Accepted

## Context
The platform should be useful in daily operations from the beginning. Teams is already a core communication environment.

## Decision
Include a lightweight Teams bot in Phase 1.

## Consequences
- Better accessibility for users who prefer quick questions in Teams
- Slightly higher Phase 1 implementation complexity
- Bot scope must stay intentionally narrow to avoid replacing the portal
