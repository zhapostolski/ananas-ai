# Ananas AI Project Memory

You are working on the Ananas AI Platform, a marketing-first internal AI system for `ai.ananas.mk`.

## Phase 1 focus
- daily marketing insights
- portal-backed structured views
- Teams channel updates
- Teams bot engagement
- executive and marketing summaries
- controlled integrations into marketing, business, and internal systems

## Architectural stance
- AWS-native
- portal-first for structured analysis
- Teams-first for communication and lightweight engagement
- specialist-first runtime
- no permanent orchestrator
- MCP-first integrations
- read-only first for core business systems
- multi-model routing

## Phase 1 specialist agents
- performance-agent
- crm-lifecycle-agent
- reputation-agent
- marketing-ops-agent
- cross-channel-brief-agent

## Phase 2 direction
- competitor intelligence
- richer lead-gen and outreach modules
- self-learning content memory
- advanced scoring and evaluation
- more autonomous workflows
- video/rendering systems
- larger agent catalog

## Non-negotiables
- Update docs when architecture changes
- Update config when behavior changes
- Run tests before claiming code is correct
- Validate outputs before persistence
- Never hardcode secrets
